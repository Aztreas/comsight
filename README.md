Comsight is a tool used to evaluate public company sentiment on social media using IBM Watson and Twitter Search API via tweets. To use, you must have both a Twitter development account (https://developer.twitter.com/), an IBM Watson Tone Analyzer Lite (https://www.ibm.com/watson/services/tone-analyzer/), and cURL (https://curl.haxx.se/dlwiz/). All resources are free.

Built for challenge lab at UC Berkeley.
